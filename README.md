在IEEE Xplorer的advanced search搜索状态下，可以搜索各种格式的文献。但面对搜索全文献的任务，存在该数据库不全的问题。

本项目针对其他开源数据库，如OpenAlex没有advanced search提出了py爬虫程序，协助开源程序的论文爬取。然而面对其他闭源数据库，还是需要手动过滤，目前没有可行的方法。
